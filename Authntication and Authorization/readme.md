# Authentication and Authhorization

### Install some packages

```
    npm init -y
    npm i express
    npm i nodemon
    npm i mongoose
    npm i dotenv
    npm i bcrypt
```

### Scripts

```
    start : node index.js
    dev : nodemon index.js
```

### Folders and files

```
    config
        -> database.js

    routes
        -> user.js

    controllers
        -> Auth.js
            -> login
            -> signup

    models
        -> users.js

```
